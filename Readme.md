Weather App (Python + Tkinter)

This project is a simple Weather Application created using Python, Tkinter, and the OpenWeatherMap API. The application allows users to select a location and view real-time weather information such as climate, description, temperature, and atmospheric pressure.


---

Features

User-friendly Graphical User Interface (GUI) built with Tkinter

Dropdown menu for selecting locations

Fetches live weather data through the OpenWeatherMap API

Displays:

Weather condition

Weather description

Temperature (Celsius)

Pressure




---

Technologies Used

Python

Tkinter (for GUI)

Requests library

OpenWeatherMap API



---

How the Application Works

1. The user opens the application.


2. Selects a location from the dropdown list.


3. Clicks the Done button.


4. The app sends a request to the OpenWeatherMap API.


5. The received weather data is processed and displayed in the GUI.




---

Installation and Setup

1. Install Python on your system.


2. Install the required library:



pip install requests

3. Save the project files in a folder.


4. Run the application using:



python main.py


---

API Key Setup

To use this application, you need an API key from OpenWeatherMap:

1. Create an account at: https://openweathermap.org/


2. Generate your free API key.


3. Replace the API key inside the code where it says:



appid=20a01944eda34a13f4a4dcecfff77197
